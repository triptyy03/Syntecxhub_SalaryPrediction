# Syntecxhub_SalaryPrediction
# Salary Prediction using Linear Regression

## Overview

This project builds a Linear Regression model to predict salary based on years of experience.

## Dataset

The dataset contains the following columns:

* YearsExperience
* Salary

## Steps Performed

1. Loaded the dataset
2. Handled missing values
3. Selected features and target variable
4. Split data into training and testing sets
5. Trained Linear Regression model
6. Made predictions on test data
7. Evaluated model using RMSE and R² score
8. Compared model performance
9. Saved the trained model

## Model Performance

* R² Score: 0.97
* RMSE: 5163

## Visualization

* Experience vs Salary (Regression Plot)
* Actual vs Predicted Salary
* Residual Plot
* Salary Distribution

## Usage

Run the notebook or script to train the model and generate predictions.
